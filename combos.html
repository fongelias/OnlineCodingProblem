Coding Problem "All Combos"

 

Write a program that produces every possible combination of a set of letters.

For example, given a string "abc":

Every possible combination of the letters in the string would be:

"abc", "acb", "bac", "bca", "cab", "cba"

(Actual formatting is show below, in the Output and Sample sections.)

 

Input

The test input file can be downloaded here.

Each line contains an input string for which all possible combinations must be computed.

For each input string:

  • Ignore whitespace

  • All the characters will be unique (you do not need to consider de-duplication of the results)

 

Output

Your program should produce an output file you can test by uploading here:

File to upload:


For each test case, your program should write the line "Case #X".

Each line that follows should contain a possible combination that can be constructed from the original input string.

  • The "Case #X" header will be on its own line. "X" should be replaced by the appropriate index of the test case.

  • Print each possible combination on its own line after the case header

  • The combinations in the output should be ordered alphabetically

 

Limits

N < 150, where N = number of test inputs

L < 8, where L = length of a test input string

 

Sample

Input:

bac

dgf

er

 

Output:

Case #1

abc

acb

bac

bca

cab

cba

Case #2

dfg

dgf

fdg

fgd

gdf

gfd

Case #3

er

re
