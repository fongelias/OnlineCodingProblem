Coding Problem "Gakkify"

 

What is a Gakky?

A "gakky" is a string containing multiple occurrences of two distinct character values in strictly alternating occurrence.

Each character must occur least twice (the minimum length of a gakky would be 4).

Here are some examples of gakkies:

ABAB

&^&^

ACACACACA

Here are some examples that are not gakkies:

Invalid example

Reason

A

Contains only one distinct character, length < 4

ABAAB

Characters to not alternate

ABCAB

Contains 3 distinct characters

ABCABC

Contains 3 distinct characters

Abab

Contains 3 distinct characters (case sensitive)

 

Your program

Write a routine called gakkify() which takes a string argument and returns a gakky contained in the string by iteratively removing all instances of a set of characters until a gakky remains.

If no gakky is possible, then the output should be an empty string.

Note: when removing a character, all instances of that character must be removed.

For example, given an input string of "abcabdacdb", gakkify() would remove all instances of 'c' and 'd', which result in a string "ababab", which is a valid gakky which gakkify() should return.

Each line contains an input string for which a gakky must be calculated.

Each input string:

  • will never contain multiple gakkies (ie. input string = "ababcdcd", which contains two possible answers: "abab" and "cdcd"), so this case may be ignored (each input will contain one and only one correct answer).

  • should exclude trailing and preceding whitespace
